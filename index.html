Récuperer et créer des projets

git init
Créer un reférentiel Git invisible ou réinitialise un existant
git clone
Cloner un référentiel dans un nouveau répertoire

Commandes de base

git add
Ajouter un fichier à l'index
git status
Afficher le statut
git diff
Affiche les différences entre commits et repo

Branching and Merging

git branch
List, create, or delete branches
git checkout
Switch branches or restore working tree files
git merge
Join two or more development histories together

Récuperer et créer des projets

git init
Créer un reférentiel Git invisible ou réinitialise un existant
git clone
Cloner un référentiel dans un nouveau répertoire

Commandes de base

git add
Ajouter un fichier à l'index
git status
Afficher le statut
git diff
Affiche les différences entre commits et repo

Branches et merges

git branch
Lister, créer ou supprimer des branches
git checkout
Changer de branchges ou restorer un fichier
git merge
Fusionner deux ou plus développement ensemble
